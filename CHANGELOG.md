## [1.0.0] - 2026-07-13

### Added

- Cross-platform `claude-gateway` setup, doctor, standalone proxy, and combined Claude launcher.
- One-time hidden credential setup and randomly generated local proxy authentication.
- Opt-in VS Code settings wizard with project-local and user scopes, non-destructive JSON
  merging, backups, token redaction, dry-run, removal, and setup integration.
- Loopback-only and telemetry-off launcher defaults.
- Extensible provider examples and complete configuration documentation.
- Tests, Ruff, pre-commit, cross-platform GitHub Actions, and Dependabot.
- Security, contribution, troubleshooting, FAQ, update, and uninstall guidance.
